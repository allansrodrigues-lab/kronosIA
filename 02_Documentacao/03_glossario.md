# Glossário

| Termo | Significado |
|---|---|
| **Lead** | Pessoa que demonstrou interesse mas ainda não agendou |
| **Lead qualificado** | Lead que respondeu perguntas-chave e tem perfil para o serviço |
| **No-show** | Cliente que marcou e não compareceu sem avisar |
| **Recompra / LTV** | Cliente que retorna para novos procedimentos |
| **Ticket médio** | Valor médio gasto por cliente por visita |
| **Funil** | Jornada: descoberta → interesse → agendamento → comparecimento → recompra |
| **Orquestrador** | Agente IA principal que decide para qual agente especialista direcionar a conversa |
| **Guardrail** | Regra que impede o agente de fazer algo proibido (ex.: dar diagnóstico médico) |
| **Handoff** | Transferência da conversa do agente para um humano |
| **Intent** | Intenção da mensagem do cliente (agendar, dúvida, reclamação, etc.) |
| **RAG** | Retrieval-Augmented Generation — agente busca na base de conhecimento antes de responder |
| **Embedding** | Representação vetorial de texto para busca semântica |
