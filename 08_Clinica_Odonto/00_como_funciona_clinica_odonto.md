# Como funciona uma clínica odontológica (guia para a Kronos)

> Material interno: para a gente saber **atender e vender** para o nicho odonto. Linguagem simples, sem enrolação.

## 1. A jornada do paciente (o caminho que ele percorre)

```
1. Primeiro contato (geralmente WhatsApp ou Instagram)
        ↓
2. Avaliação / consulta inicial  ← aqui o dentista examina e diz o que precisa
        ↓
3. Plano de tratamento + orçamento  ← "você precisa de X, Y, Z; custa R$ tanto"
        ↓
4. Aprovação do orçamento (o momento crítico — muitos somem aqui)
        ↓
5. Sessões de tratamento (1 ou várias, dependendo do caso)
        ↓
6. Alta + manutenção/retorno (limpeza a cada 6 meses)
```

**Onde a clínica perde dinheiro:** entre o passo 3 e 4 (orçamento não aprovado) e nas faltas em qualquer sessão. É exatamente onde a Kronos entra.

## 2. As especialidades (quem faz o quê)

| Especialidade | O que trata | Exemplos |
|---|---|---|
| **Clínico geral / dentística** | O dia a dia | Limpeza, restauração (obturação), avaliação |
| **Endodontia** | "Canal" — trata a raiz/nervo do dente | Tratamento de canal |
| **Ortodontia** | Alinhamento dos dentes | Aparelho fixo, aparelho estético, alinhadores invisíveis |
| **Implantodontia** | Repõe dentes perdidos | Implante + coroa |
| **Prótese** | Substitui dentes | Coroa, ponte, dentadura |
| **Odontologia estética** | Estética do sorriso | Clareamento, lentes/facetas |
| **Periodontia** | Gengiva | Tratamento de gengivite, raspagem |
| **Odontopediatria** | Crianças | Atendimento infantil |

> Uma clínica pequena costuma ter 1 clínico geral que faz o básico e **encaminha** os casos complexos para especialistas (próprios ou parceiros).

## 3. Particular x Convênio

- **Particular:** o paciente paga direto. Margem maior, mas precisa captar e fechar orçamento.
- **Convênio (plano odontológico):** o paciente tem um plano (Amil Dental, OdontoPrev, etc.); a clínica recebe do convênio (valor menor, mas volume).
- **Plano próprio / clube de assinatura:** a clínica cria seu *próprio* plano de mensalidade — **renda recorrente** (é o que mais interessa pra nós e pro dono da clínica).

## 4. Termos que você vai ouvir (glossário rápido)

| Termo | Tradução |
|---|---|
| Profilaxia | Limpeza |
| Restauração / obturação | Tampar o dente cariado |
| Endodontia / canal | Tratar a raiz do dente |
| Exodontia | Extração (arrancar o dente) |
| Siso / terceiro molar | O dente do fundo que costuma extrair |
| Prótese | Dente artificial |
| Coroa | "Capa" que cobre o dente |
| Faceta / lente de contato | Lâmina fina colada no dente (estética) |
| Aparelho ortodôntico | Aparelho dos dentes |
| Recall / retorno | Chamar o paciente de volta (ex: limpeza a cada 6 meses) |
| Anamnese | Ficha de saúde preenchida na 1ª consulta |

## 5. O ciclo de recompra (a mina de ouro)

- **Limpeza a cada 6 meses** → todo paciente vira recorrente se for lembrado.
- **Ortodontia** → mensalidade fixa por 18-24 meses.
- **Plano de assinatura** → receita mensal previsível.

A Kronos automatiza o "lembrar de voltar" — que é onde a clínica deixa dinheiro na mesa por não ter quem faça.
